# 🔦 内存分析知识点（未完）

## 1.常用指令

主要的格式是

```
volatility -f [imame] --profile=[profile][plugin]
volatility -f [对象] --profile=[操作系统][插件参数]
```



### ①查看操作系统

在分析之前，先判断镜像信息，分析是哪个操作系统，使用imageinfo就可以。

```
volatility -f xxxx.vmem imageinfo
```
